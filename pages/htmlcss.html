St. Anthony Academy Website
About this project

I created this website back in 2014 and I was the designer and the programmer in the team. The idea was to create a website for St. Anthony Academy that will help the students and the faculty to get the latest school updates and announcements.


Please note
After downloading the project in a zip folder, make sure to extract it first so it would work properly and then open the "index.html".

All the pages in the project files are in ".php" format so in order to use all the functionality of this website, you need to upload the source files first into a localhost but I've also created an HTML version for review purposes.

Wafer Puzzle
About this project

This project was a part of WordPress development and the customer asked for an interactive puzzle for their organization. The idea was when a user hover their cursor to one of the puzzle piece, it will decrease its opacity and when they click it, a popup will open showing the description of that piece. The most challenging part in this project was when I had to put each pieces one by one to their correct placement.


Please note
After downloading the project in a zip folder, make sure to extract it first so it would work properly and then open the "Wafer.html".
